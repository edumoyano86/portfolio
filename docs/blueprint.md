# **App Name**: EduMoyano Portfolio

## Core Features:

- Homepage: Landing page with a brief introduction and overview of the portfolio.
- Studies Section: Dedicated page to showcase educational background, degrees, certifications, and relevant coursework.
- Projects Showcase: Display completed projects with descriptions, images, and links to live demos or repositories.
- Skills Section: Highlight technical skills, programming languages, tools, and frameworks.
- Work in Progress: A view to show the current progress of projects

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey professionalism and creativity.
- Background color: Very light grey (#F0F0F0), providing a clean and unobtrusive backdrop.
- Accent color: A bright teal (#008080) to draw attention to important elements and calls to action.
- Body font: 'Inter', a sans-serif font with a modern, neutral look suitable for readability.
- Headline font: 'Space Grotesk', a sans-serif font with a techy feel.
- Use a consistent set of professional and modern icons for navigation and highlighting key skills and technologies.
- A clean and well-organized layout with clear sections for easy navigation and a visually appealing presentation.
- Subtle animations on hover or scroll to add a touch of interactivity and visual interest without being distracting.
